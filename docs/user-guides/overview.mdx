## What the Control Tower is for

The Control Tower is your operational hub. It helps you:

- Monitor active onboarding workflows and their stages
- See real-time activity across applicants and workflows
- Respond to alerts and human-in-the-loop approvals
- Track agent performance and workflow health

## Main navigation

Use the left sidebar to move between sections:

- **Overview**: dashboard summary, active workflows, recent activity
- **Applicants**: manage applicants and start onboarding
- **Workflows**: track all workflows, take action, and view details
- **Agents**: monitor your external agent fleet

## Dashboard sections at a glance

The overview page is split into a summary grid and live activity.

### Summary cards

You will see cards for:

- **Active Workflows**: number of workflows currently running
- **Pending Approvals**: workflows awaiting human input
- **Total Applicants**: count of all applicants in the system
- **Completion Rate**: a high-level success indicator (if configured)

- **Client** and workflow ID
- **Stage** (1â€“4) with a progress indicator

Use sorting on the table headers to prioritise urgent work.

### Recent Activity feed

This feed lists the latest workflow events, including:

- Stage changes
- Agent dispatches and callbacks
- Human overrides
- Errors and timeouts

## Workflow stages

The onboarding process follows a consistent 4-stage path:

1. **Applicant Capture**: record basic client details and start the workflow
2. **Quotation**: collect core agreement data and service configuration
3. **Verification**: complete bank assessment and FICA document checks
4. **Integration**: final handover and operational readiness

## Notifications

Notifications appear in the top bar and alert you to:

- Items awaiting approval
- Timeouts or errors that require action
- Completed stages or workflows

For step-by-step instructions, see `User guides > Notifications`.
