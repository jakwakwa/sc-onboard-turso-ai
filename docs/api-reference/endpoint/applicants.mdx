## List Applicants

<Segment>
  <SegmentTitle>GET /api/applicants</SegmentTitle>
  <SegmentDescription>Retrieve a paginated list of all applicants.</SegmentDescription>
</Segment>

### Response

```json
{
  "data": [
    {
      "id": 1,
      "companyName": "Acme Corp",
      "status": "new",
      "riskLevel": "green"
    }
  ]
}
```

## Get Applicant

<Segment>
  <SegmentTitle>GET /api/applicants/{id}</SegmentTitle>
  <SegmentDescription>Retrieve details for a specific applicant.</SegmentDescription>
</Segment>

## Create Applicant

<Segment>
  <SegmentTitle>POST /api/applicants</SegmentTitle>
  <SegmentDescription>Register a new applicant in the system.</SegmentDescription>
</Segment>

### Request Body

<ParamField body="companyName" type="string" required>
  The legal name of the company.
</ParamField>

<ParamField body="contactName" type="string" required>
  Full name of the primary contact person.
</ParamField>

<ParamField body="email" type="string" required>
  Email address for notifications and correspondence.
</ParamField>

<ParamField body="phone" type="string" required>
  Contact phone number.
</ParamField>

<ParamField body="industry" type="string" required>
  Industry sector (e.g., "Retail", "Manufacturing").
</ParamField>

<ParamField body="employeeCount" type="integer" required>
  Number of employees.
</ParamField>

<ParamField body="estimatedVolume" type="string">
  Estimated monthly transaction volume (e.g "R500,000").
</ParamField>

<ParamField body="notes" type="string">
  Additional notes about the applicant.
</ParamField>

### Response

```json
{
  "applicant": {
    "id": 12,
    "companyName": "Acme Corp",
    "status": "new",
    "riskLevel": null,
    "createdAt": "2023-11-21T10:00:00Z"
  },
  "workflow": {
    "id": 45,
    "status": "pending",
    "stage": 1
  }
}
```
